Description

This is the Wasm build of the WebGPU + SDL2 starter template. Using this template you can write a single C++ application that uses SDL2 and WebGPU and compiles to both native and Web platforms. Grab the starter template here: github.com/Twinklebear/webgpu-cpp-wasm.
